# ngx-ark

Une bibliothèque de composants UI pour l'écosystème Angular, entièrement documentée avec Storybook.

## À propos du projet

**ngx-ark** a été conçue pour offrir des bases solides, maintenables et scalables pour le développement d'applications web modernes. Chaque composant est développé de manière isolée,testé et exposé à travers une interface **Storybook** interactive. Cela garantit une documentation vivante, des tests visuels simplifiés et une adoption fluide pour les développeurs.
